# beatly claude-code marketplace

Claude Code plugin marketplace for [Beatly](https://beatly.dev).

> **This repository is auto-generated.** Don't commit to it by hand — the `plugins/beatly/` directory is replaced by CI on every Beatly release. Source lives at [`getbeatly/beatly`](https://github.com/getbeatly/beatly).

## Install

```text
/plugin marketplace add getbeatly/claude-code
/plugin install beatly@beatly
```

Then restart Claude Code.

## Requirements

Beatly needs SuperCollider installed system-wide, with `scsynth` and `sclang` on `PATH`.
